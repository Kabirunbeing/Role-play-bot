## New Design System

### Color Palette
**Base Colors:**
- Pure Black: `#000000` - Main background
- Pure White: `#ffffff` - Text and borders
- Off Black: `#0a0a0a` - Card backgrounds
- Dark Gray: `#1a1a1a` - Input fields

**Vibrant Accent Colors:**
- Neon Green: `#00ff41` - Primary buttons, Friendly personality
- Neon Cyan: `#00f0ff` - Wise personality, focus states
- Neon Yellow: `#ffff00` - Sarcastic personality
- Neon Pink: `#ff006e` - Cheerful personality, danger actions
- Neon Purple: `#b026ff` - Dark personality

### Typography
- Display Font: **Space Grotesk** - Headers, character names
- Body Font: **Inter** - Body text, descriptions
- Mono Font: **Fira Code** - Timestamps, stats

### Components Styled

1. **Buttons**
   - Primary: Neon green with glow effect on hover
   - Secondary: Pure white background
   - Outline: White border, fills on hover
   - Danger: Neon pink with glow

2. **Cards**
   - Dark gray background
   - White/10% opacity borders
   - Hover states with neon accent borders

3. **Input Fields**
   - Dark gray background
   - White/20% borders
   - Neon cyan focus state with glow

4. **Custom Features**
   - Neon glow shadows on interactive elements
   - Smooth scale transforms on hover
   - Animated typing indicators with multi-color dots
   - Custom scrollbar styling

## Design Philosophy

- **High Contrast**: Black & white base for maximum readability
- **Vibrant Accents**: Neon colors for interactivity and personality
- **Modern Aesthetic**: Clean typography, generous spacing
- **Playful Details**: Emoji avatars, color-coded personalities, glow effects
- **Smooth Interactions**: Transitions, hover states, scale effects
